## 🚀 Como Usar

### Localmente

1. **Clone ou baixe o projeto:**
```bash
git clone https://github.com/rodrigo-ed/my-portfolio.git
cd portfoliosegundo
```

2. **Abra em um servidor local:**
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server -p 8000

# PHP
php -S localhost:8000
```

3. **Visite:** `http://localhost:8000`

## ⚙️ Configuração

### Alterar Dados Pessoais

**Em index.html:**
```html
<!-- Nome -->
<span itemprop="name">Seu Nome</span>

<!-- Email -->
<a href="mailto:seu@email.com">seu@email.com</a>

<!-- WhatsApp -->
<a href="https://wa.me/5585987654321">+55 (85) 98765-4321</a>

<!-- Redes Sociais -->
<a href="https://github.com/seu-usuario">GitHub</a>
<a href="https://linkedin.com/in/seu-usuario">LinkedIn</a>
```

### Customizar Cores

**Em css/style.css:**
```css
:root {
    --primary: #3b82f6;           /* Cor primária */
    --primary-dark: #2563eb;      /* Cor primária (escura) */
    --primary-light: #1a6bff;     /* Cor primária (clara) */
    /* ... outras variáveis */
}
```

### Configurar Domínio

**Em index.html (meta tags):**
```html
<link rel="canonical" href="https://seudominio.com.br/" />
<meta property="og:url" content="https://seudominio.com.br/" />
```

**Em sitemap.xml:**
```xml
<loc>https://seudominio.com.br/</loc>
```

**Em robots.txt:**
```
Sitemap: https://seudominio.com.br/sitemap.xml
```
