# 🚀 Portfólio de Rodrigo Eduardo — Desenvolvedor Web & Suporte Técnico

Portfólio profissional moderno e responsivo de Rodrigo Eduardo, apresentando seus serviços de desenvolvimento web e suporte técnico. Desenvolvido com HTML5, CSS3 e JavaScript vanilla, sem dependências externas.

**Status:** ✅ **PRONTO PARA DEPLOY**

## ✨ Características

- 🎨 **Design Moderno** - Interface limpa e profissional

- 📱 **Totalmente Responsivo** - Funciona em todos os dispositivos
- ♿ **Acessível** - WCAG 2.1 AA compliance
- 🔍 **SEO Otimizado** - Meta tags, Schema.org markup, sitemap

- 🔐 **Seguro** - Headers de segurança, HTTPS ready
